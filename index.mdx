## Quick Example

Here's how simple it is to get today's soccer events with betting recommendations:

<CodeGroup>
```bash cURL
curl -H "Authorization: Bearer sk_live_your_api_key" \
  "https://api.mrdoge.co/odds/events?sport=soccer&status=upcoming&status=live&locale=en"
```

```javascript JavaScript
const response = await fetch("https://api.mrdoge.co/odds/events?sport=soccer&status=upcoming&status=live&locale=en", {
  headers: {
    Authorization: "Bearer sk_live_your_api_key",
  },
});
const events = await response.json();
console.log(`Found ${events.length} upcoming matches`);
```

```python Python
import requests

headers = {'Authorization': 'Bearer sk_live_your_api_key'}
response = requests.get(
    'https://api.mrdoge.co/odds/events',
    params={'sport': 'soccer', 'status': ['upcoming', 'live'], 'locale': 'en'},
    headers=headers
)
events = response.json()
print(f"Found {len(events)} upcoming matches")
```
